# WP Portfolio Starter

A clean WordPress-style portfolio starter template for developers and freelancers. This repo is designed as a simple front-end prototype that can be converted into a WordPress theme.

## Features

- Responsive portfolio layout
- Services section
- Project cards
- Contact CTA
- Clean HTML, CSS, and JavaScript

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Folder Structure

```bash
wp-portfolio-starter/
├── index.html
├── assets/
│   ├── css/style.css
│   └── js/main.js
└── README.md
```

## How to Use

Open `index.html` in your browser and customize the content, colors, and portfolio projects.
